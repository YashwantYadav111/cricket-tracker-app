# Cricket React Pro
Pure React cricket tracker. PHP/MySQL removed.

## Run
```bash
npm install
npm run dev
```

## Live API
Create `.env` from `.env.example` and add RapidAPI key:
```bash
VITE_RAPIDAPI_KEY=your_key
VITE_RAPIDAPI_HOST=cricbuzz-cricket.p.rapidapi.com
```

If API key is missing, app uses mock cricket data automatically.
